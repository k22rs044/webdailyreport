# MySQL 開発環境

## 使い方

### 1. 環境変数の設定
プロジェクトのルートに`.env`ファイルを作成します。`.env.example`の名前を変更するのが簡単です。

作成した`.env`ファイルを開き、MySQLのrootパスワードを設定してください。

例

```
MYSQL_ROOT_PASSWORD="password"
```

### 2. 起動
以下のコマンドを実行すると、コンテナのビルドが開始され、バックグラウンドで起動します。

```bash
docker-compose up --build -d
```

### 3. 停止
コンテナを停止し、関連するネットワークなどを削除するには以下のコマンドを実行します。

```bash
docker-compose down
```

## 各サービスについて

### `db` (MySQL データベース)
- **注意:** このデータベースコンテナは、ポートを外部に公開していません。
- そのため、お使いのPCにインストールされたGUIツール（TablePlus, DBeaverなど）から直接接続することはできません。
- データベースへの接続は、`app`コンテナなど、同じDockerネットワーク内のサービスからのみ可能です。
  - **ホスト名**: `db`
  - **ユーザー名**: `root`
  - **パスワード**: `.env`で設定した `password`
  - **データベース名**: `my_database`

### `phpmyadmin`
- Webブラウザからデータベースを操作するためのツールです。
- **URL**: [http://localhost:8081](http://localhost:8081)
- ログイン情報:
  - **サーバー**: `db`
  - **ユーザー名**: `root`
  - **パスワード**: `.env`で設定した "password"
